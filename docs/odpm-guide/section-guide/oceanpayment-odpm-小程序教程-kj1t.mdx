#   ODPM路径配置说明

- **配置路径：**
ODPM>应用>小程序设置>新增账号

![图片](/img/help-center/oceanpayment-odpm-小程序教程-kj1t/T7wXb5qypohX4WxOmWpcMPVdnpc.png)

![图片](/img/help-center/oceanpayment-odpm-小程序教程-kj1t/VbyebiIrBosOWQx7yalcCnzLn9b.png)

- **配置说明：**
2.1：填写对应手机号进行绑定，创建新的用户名跟岗位名称且启用即可

2.2：该板块创建的小程序账号信息不影响其他板块的角色信息

- **注意事项：**
3.1：账号创建后需以绑定的手机号进行登陆

3.2：目前同商户号下支持创建十个小程序账号

#  小程序登陆教程

# **小程序登陆教程**

- 登陆界面说明：目前登陆方式为微信绑定对应手机号一键登陆，如微信未绑定对应手机号，可以点击管理号
码进行绑定操作，该界面退出后，小程序后台停留时间为十分钟，十分钟后需重新登陆。

![图片](/img/help-center/oceanpayment-odpm-小程序教程-kj1t/HtRIbikmJoBPDcxZvy4cr4ejnKc.png)

- **板块说明：**目前一共分为三大板块-首页、消息、我的
**首页：**目前分为五个二级板块：待办事项、账户资金明细、交易明细、交易划款、可视化图表。

![图片](/img/help-center/oceanpayment-odpm-小程序教程-kj1t/GU7xbC9TMoZhiZxIDr8cUR0pnHh.png)

- **待办事项**：该板块主要用于查看账户今日成功笔数与成功金额，以及待处理的待授权、拒付、争议、伪冒、调单笔数。此数据为实时更新，需商户登陆PC端进行操作。
![图片](/img/help-center/oceanpayment-odpm-小程序教程-kj1t/WDFFbfSrKopq9uxthxkcKhmgnsb.png)

- **账户资金明细**：该板块主要用于账户资金情况查看，可查看板块为：账户结算方式、时间周期、可结算金额、累计未结算金额、累计划款金额，此数据有三十分钟的数据延迟。
[附件: 附件 — 无法提取下载链接，请在飞书中以"附件"方式插入]

![图片](/img/help-center/oceanpayment-odpm-小程序教程-kj1t/PvI2bk08WoPE9WxMMPrcjUDhnEA.png)

![图片](/img/help-center/oceanpayment-odpm-小程序教程-kj1t/Tewob2blro7ODKxF7Ddcpioen6e.png)

- **交易明细**：该板块主要用于账户交易数据情况查询，查询时间仅限一年的数据，可查询板块包含：账户订单号、支付ID、时间类型、划款批次号、交易币种、记账币种、交易类型、卡种、是否上传物流单号、妥投状态、交易划款、保证金划款、终端号，此部分数据为实时更新。
[附件: 附件 — 无法提取下载链接，请在飞书中以"附件"方式插入]

![图片](/img/help-center/oceanpayment-odpm-小程序教程-kj1t/Bp38bw028oRfoWxzHTfcXTBynye.png)

![图片](/img/help-center/oceanpayment-odpm-小程序教程-kj1t/YwVmbDP63otRsOx1jQOclKpZnvg.png)

- **交易划款**：该板块主要用于账户交易划款数据的查询，查询时间仅限一年的数据，可查询板块包含：划款批次号、时间类型、币种、划款类型，此部分数据与ODPM数据同步。
[附件: 附件 — 无法提取下载链接，请在飞书中以"附件"方式插入]

![图片](/img/help-center/oceanpayment-odpm-小程序教程-kj1t/FYWcbU2PpopWOKxbBOBcxJ15nQh.png)

![图片](/img/help-center/oceanpayment-odpm-小程序教程-kj1t/Voo0btIEOocXesxDlq0ceRVnnFg.png)

- **可视化图表**：该板块可用于查询不同终端号、不同的图表类型，可查询图标类型：整体交易分析、各发卡国家交易分布、整体成功率分析、整体失败原因、整体拒付原因、整体拒付/伪冒和退款(区分卡种)、整体拒付伪冒退款分析、整体物流情况，次部分图标类型可自行选择展示那部分的数据，时间类型选择支持多个类型。

- **消息板块：**用于各类消息的通知，包含：系统公告、拒付通知、伪冒通知、调单通知，支持仅显示未读消息。
![图片](/img/help-center/oceanpayment-odpm-小程序教程-kj1t/AHjlbXgLloM2KVxzq2Gcu4UYnrb.png)

- **我的板块：**主要用于小程序账号的主账号名称、用户名以及绑定手机号，同时会展示部门客服电话、客服邮箱以及对应商户号的所属客关手机号以及企微名片，名片支持长按识别，小程序账号信息仅支持PC端进行修改。
