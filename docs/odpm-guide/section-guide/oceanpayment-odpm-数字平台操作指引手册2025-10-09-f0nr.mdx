## 操作设置

操作设置一共分为三个板块：**角色、数据权限、管理员。**

- **角色**
点击【角色】-【添加角色】，则可以根据账户使用者的角色或者部门职能对角色进行设置。在设置页面输入相应的信息并勾选对应的权限之后，点击“提交”，则角色设置成功。然后进入【角色设置】的页面，则可以查看当前账户所设置的所有角色的列表，并可以针对某个角色的信息或者权限进行修改，或者删除某个角色。

# 风控管理（PC/H5)

- 账户订单号/支付 ID
- 默认【推送时间】:为警报扫描后，高风险订单的推送时间
- 交易时间:警报订单的交易时间
- 处理时间:商户对警报订单进行退款操作的时间
- 终端号:可以选择同时选择多个终端号
- 邮箱:区分大小写，精准查询
- IP 地址、卡号前6位、卡号后 4位、交易币种:精准查询
- 是否拒付:订单已发生拒付则为是，否则为否
- 是否伪冒:订单已发生伪冒则为是，否则为否
- 是否退款:订单已完成退款则为是，否则为否
- 是否 3D:3D 交易则为是，否则为否
- 是否预授权: 预授权交易则为是，否则为否
- 处理状态:【高欺诈风险警报递知】的订单不管是通过 Alert Rules 自动授权完
- 风险状态:高风险交易、风险预警交易
- 高风险交易:黑名单交易:此订单的信息因“盗卡、欺诈拒付、伪冒、批量
- 风险预警交易:高金额订单:高金额订单提醒(如订单金额900USD及以
- 授权撤销:信用卡预授权交易，且【风险状态=风险预警交易】，自动【授权
- 授权完成:信用卡预授权交易，且【风险状态=风险预警交易】，自动【授权
- 授权撤销:信用卡预授权交易，且【风险状态=高风险交易】，自动【授权撤
- 授权完成:信用卡预授权交易，且【风险状态=高风险交易】，自动【授权完
