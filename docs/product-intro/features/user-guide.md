# ODPM帮助中心使用指南

## 一、功能介绍

帮助中心用于管理企业文档和发布流程，支持内容管理、目录设置、飞书文档同步、发布部署等核心功能。

## 二、快速开始

### 2.1 登录系统

打开系统地址，使用账号登录后即可进入运营工作台。

### 2.2 新建文档

进入「内容管理」页面，点击「新建文档」按钮，填写标题、选择分类、编写正文内容后保存。

### 2.3 发布流程

文档保存后进入草稿状态，管理员可在「发布中心」进行内容检查和发布操作。

## 三、核心功能

- **运营工作台**：查看文档统计数据、最近更新文档、最近发布任务
- **内容管理**：文档的创建、编辑、删除和筛选
- **目录设置**：管理帮助中心的目录结构
- **发布中心**：管理发布任务、查看发布日志
- **系统配置**：配置仓库地址、环境地址等系统参数
