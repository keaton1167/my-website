# 常见问题解答

## 1. 如何新建文档？

点击内容管理页面的「新建文档」按钮，填写标题、选择分类后即可开始编写内容。

## 2. 文档保存后如何发布？

文档保存后默认进入草稿状态。管理员可在发布中心进行内容检查，通过后可发布到测试环境或正式环境。

## 3. 如何修改目录结构？

进入「目录设置」页面（需要管理员权限），可以新建、编辑、排序和启用/禁用目录。

## 4. 飞书文档如何同步？

进入「导入工具」页面，配置飞书文档映射关系后，可以手动或定时同步飞书文档到帮助中心。

## 5. 发布失败如何处理？

查看发布任务的日志信息，根据错误提示修复后重新发布。严重问题请联系系统管理员。
