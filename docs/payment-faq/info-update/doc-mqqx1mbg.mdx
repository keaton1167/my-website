# 附件预览测试

本页用于验证帮助中心对 PDF / PPTX / XLSX 附件的展示效果。

附件可在页面内预览，同时保留原文件下载入口。

## PDF

<a href="/files/help-center/doc-mqqx1mbg/%5BODPM%C2%A0Digital%C2%A0Platform%5D%C2%A0-%C2%A0Whitelist%C2%A0Operation%C2%A0Manual%C2%A02025-07-25.pdf">[ODPM Digital Platform] - Whitelist Operation Manual 2025-07-25.pdf</a>

PDF 支持页面内预览，适合保留原始手册。

如需右侧目录跳转和全文搜索，建议转成飞书云文档正文。

## PPTX

<a href="/files/help-center/doc-mqqx1mbg/Pay%20By%20Link%20Guideline%20-%20Oceanpayment%20-EN.pptx">Pay By Link Guideline - Oceanpayment -EN.pptx</a>

PPTX 支持生成封面/幻灯片图片预览，并保留原文件下载。

适合培训材料、演示资料、补充说明。

## Excel

<a href="/files/help-center/doc-mqqx1mbg/%E6%B5%8B%E8%AF%95%E8%A1%A8%E6%A0%BC.xlsx">测试表格.xlsx</a>

Excel 支持生成表格预览，并保留原文件下载。

适合模板、清单、对账表等资料。
